# Quick Start — NestJS + PostgreSQL Docker Setup

## Prerequisites

- Docker (≥ 20.x)
- Docker Compose (≥ v2.x)
- `.env` file in project root
- Ports `3000` and `5432` open

---

## 1 Clone Repository

```bash
git clone <repository-url>
cd <project-folder>
```

## 2 Create .env

- Copy given .env file

## Build & Start Containers

```bash
docker compose up --build -d
```

- To check if a container is running:

```bash
docker ps
```
